1. Muse 는 무엇인가

Muse Agent 는 if-only Echo 서비스의 PC 동반 에이전트입니다.

가장 큰 역할은 Echo Note (잔잔한 물결) 의 강화 privacy 보관 경로를 제공하는 것이에요.

2. 강화 privacy 모델

Echo Note (잔잔한 물결) 본문은 다음 경로로만 흐릅니다:

  • 보관: 사용자 PC 의 muse-agent SQLite
  • 발송 시점: muse 가 echo 의 외부 API 로 가공본을 전달
  • echo 의 송신 로그: 메타 정보만 잔존

OAuth2 자격증명은 Windows DPAPI 로 암호화된 secure-config 파일에 보관돼요.

3. 주요 사용 흐름

📨 시나리오 A — Echo Note 보관·발송

  1. echo popup 진입
  2. 대시보드에서 보관
  3. 미리보기 → 보관
  4. 마침 처리 → 자동 발송
  5. 또는 즉시 수동 발송

🌡️ 시나리오 B — Alive 모니터링 + 자동 정리

  1. Alive-Check 폴링
  2. EXPIRED 임계 시 cleanup
  3. 매 cycle 자동 만료 purge

설정은 시스템 패널에서.

🗑️ 시나리오 C — 휴지통 수동 관리

  1. 시스템 패널에서 조회
  2. 복원 또는 즉시 삭제
  3. Purge Now 강제 실행
