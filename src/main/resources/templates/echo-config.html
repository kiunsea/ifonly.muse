⚠ 에이전트 재시작 필요

OAuth2 토큰 발급 설정(URL / Client ID / Secret)은 재시작 후 적용됩니다.

Windows 서비스로 실행 중인 경우 — 관리자 권한 명령 프롬프트에서:

sc stop muse-agent && sc start muse-agent

또는 WinSW 설치 폴더에서:

muse-agent-service.exe restart

서비스가 아닌 일반 실행이라면 실행 중인 muse-agent 창을 닫고 배치 파일로 재시작하세요.

실행 중인 muse-agent 프로세스를 종료 후 재시작하세요.
